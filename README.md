# BabyBlue

### Problem Description
Breaking Bad is a widely known American television series that depicts the series of life transformations of a high school chemistry teacher. In the show, the main character, White, is especially skilled at making a delicious dessert called Blue Sky.

In this assignment, we model the situation by considering the selling prices of different packages of Blue Sky.

The array prices represents the prices of different package sizes of Baby Blue.
The shop owner, Fring, sets a required monthly sales target of amount.
Your task is to determine the number of distinct combinations of package prices that sum up exactly to amount.

Each price in prices may be used an unlimited number of times.
The order of selection does not matter
(for example, using prices 2 + 3 is considered the same as 3 + 2).
If it is not possible to reach the target amount, return 0.
Return an integer representing the total number of distinct combinations that can achieve the target amount.
